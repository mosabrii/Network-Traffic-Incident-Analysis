# Evidence

This folder contains supporting evidence used during the investigation of the security incident.

The files included here were reviewed to understand how the attack occurred, how the malicious behavior was observed, and what network activity confirmed the incident.

## Included Evidence

- **tcpdump-traffic-log.pdf**
  - Network traffic capture used to analyze DNS and HTTP communication.
  - Shows normal access to the legitimate website followed by redirection to a malicious domain.
  - Confirms suspicious network behavior related to the incident.

## Purpose of Evidence

The evidence in this folder supports:
- Incident analysis and validation
- Timeline reconstruction
- Identification of affected protocols and destinations

These files are provided for reference and documentation purposes only.
